# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please open a private security advisory or contact the maintainer directly.

Do not disclose vulnerabilities publicly until they have been addressed.

## Supported Versions
Security updates apply to the latest stable release of AAPP-MART.

## Scope
This project does not include exploit code.  
Reports should focus on:
- Code weaknesses
- Dependency vulnerabilities
- Logic flaws
- Misconfigurations
